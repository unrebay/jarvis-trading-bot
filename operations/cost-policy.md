# Cost Policy

## Claude API usage
- Структурирование знаний: батчить файлы, не гонять по одному
- Анализ графиков: только по запросу пользователя
- Telegram бот: отвечать только при обращении (не polling без причины)

## VPS
- Минимальный инстанс пока нет нагрузки
- Масштабировать только при > 10 активных пользователей

## GitHub
- Хранить только текст и JSON
- Медиа (видео, изображения) — хранить отдельно (S3/B2/локально)
- Не коммитить тяжёлые файлы

## Media storage
- Скриншоты: локально на iMac + sync на VPS по необходимости
- Видео: только на iMac, не синхронизировать
- Кадры из видео: только релевантные, после аннотации
