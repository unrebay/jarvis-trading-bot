# Инструменты для торговли (Trading Platforms)

## Торговые платформы

В DT курсе используются следующие торговые платформы:

### TradingView
- Основная платформа для анализа графиков
- Веб-версия: tradingview.com
- Поддерживает все инструменты технического анализа
- Идеально для рисования уровней, структуры, FVG
- Доступны скрипты и индикаторы

### Ctrader | MatchTrader
- Платформа для исполнения сделок
- Используется у большинства форекс-брокеров
- Профессиональный интерфейс для трейдинга
- Поддерживает автоматические стратегии

### Metatrader (MT4/MT5)
- Самая популярная торговая платформа в мире
- Доступна в веб-, десктопной и мобильной версиях
- Поддерживает форекс, CFD, крипто
- Широкий выбор брокеров с поддержкой MT4/MT5

### Metatrader (Phone version)
- Мобильная версия MetaTrader для iOS и Android
- Позволяет торговать с телефона
- Основные функции: открытие/закрытие сделок, мониторинг позиций, графики

## Рекомендации

Для изучения курса DT рекомендуется:
1. TradingView — для анализа и обучения
2. Ctrader или MT5 — для реального трейдинга

Канал instruments на сервере содержит видео-туториалы по настройке каждой платформы.
