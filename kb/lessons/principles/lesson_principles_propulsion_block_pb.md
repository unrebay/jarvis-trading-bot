# Propulsion Block (PB)

## Определение

**Propulsion Block (PB)** — это блок или свеча, сформированная **внутри Ордер Блока в момент теста**. В последствии данная свеча берет на себя роль ценовой поддержки для более высокого/низкого ценового движения.

## Логика формирования

Propulsion Block формируется в момент, когда цена тестирует Order Block. Внутри этого теста образуется свеча/блок, который в дальнейшем выступает как зона поддержки/сопротивления для продолжения движения.

## Применение

- Используется как уточнённая точка входа внутри Order Block
- Позволяет получить более точный уровень для стопа
- Является зоной спроса/предложения наряду с Breaker Block, Mitigation Block, Rejection Block
- Применяется в контексте торговой модели как POI

## Связь с другими концептами

- Формируется **внутри OB** (в отличие от Breaker, который OB прошит FVG)
- Даёт более точную зону внутри старшего POI
- Используется совместно со структурой и контекстом
