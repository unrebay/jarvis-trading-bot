# Fair Value Gap (FVG) — Ценовой Дисбаланс

## Определение

**Imbalance / FVG (Fair Value Gap)** — это ценовой дисбаланс. Диапазон на графике, где было заложено мало ликвидности.

Imbalance подразумевает сильное импульсное движение свечи, которое цена почти **всегда возвращается перекрывать**.

**Аналогия**: представьте, что красите стену валиком. Провели сверху вниз — остались пустоты. Теперь нужно провести снизу вверх — заполнить пустоты. Аналогичный принцип с имбалансом.

## Как правильно отмечать FVG?

Нужно найти **три свечи**, одна из которых будет импульсная. Используя инструмент «Коррекция по Фибоначчи» натягиваем **от фитиля первой свечи до фитиля последней**.

FVG = диапазон между хвостом (фитилем) первой и третьей свечи, где тело второй свечи создало дисбаланс.

## Типы перекрытия

- **Full Fill** — полное перекрытие (100%)
- **50% / C.E (Consequent Encroachment)** — перекрытие до 50% — наиболее частый вариант
- **IOFED** — минимальное перекрытие (1%+), указывает на начало ребалансировки

## Применение

Imbalance (FVG) используется:
- **Для определения контекста** — на старшем ТФ выступает как зона интереса (POI)
- **В модели для входа** — ищем реакцию цены при касании зоны FVG
- Внутри FVG на старшем ТФ можно искать факторы для набора позиции на младшем

## SIBI / BISI

- **BISI** = Buyside Imbalance SellSide Inefficiency — FVG при апмуве (бычий FVG)
- **SIBI** = Sellside Imbalance BuySide Inefficiency — FVG при даун-муве (медвежий FVG)
