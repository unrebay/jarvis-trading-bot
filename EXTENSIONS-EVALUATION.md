## 🟠 ВЫСОКИЙ ПРИОРИТЕТ (Установить в первую очередь)

### 3. **GitHub MCP** ⭐⭐⭐⭐ (9/10)
- **Статус:** ❌ Не найден в registry
- **Функция:** Commit, push, pull, PR, issues прямо из Claude
- **Для JARVIS:** Вместо `git push` с Mac — можно через Claude
- **Полезность:** ВЫСОКАЯ — ускорит workflow (сейчас юзим `git show`)
- **Проблема:** Нужна ручная установка (`https://api.githubcopilot.com/mcp`)
- **Действие:** ⚠️ Установить вручную в Claude Desktop Settings
  ```
  Claude Desktop → Settings → Extensions (gear icon) →
  Add MCP Server → https://api.githubcopilot.com/mcp
  ```


```
3. Look for "Extensions" or "MCP Servers"
4. Click "+ Add MCP Server"
5. Paste: https://api.githubcopilot.com/mcp
6. Click "Connect"
7. Restart Claude
```
